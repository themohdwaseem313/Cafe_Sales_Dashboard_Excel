# ☕ Cafe Sales Dashboard (Microsoft Excel)

## 📌 Project Overview

This project demonstrates the creation of an interactive **Cafe Sales Dashboard** using **Microsoft Excel**. The objective was to clean, analyze, and visualize sales data to generate meaningful business insights.

The original dataset contained **10,000+ transaction records**. After performing data cleaning and removing inconsistent or missing records, the final dataset consisted of **9,000+ clean records** for analysis.

---

## 📂 Dataset Information

The dataset includes the following columns:

- Transaction ID
- Item
- Quantity
- Price Per Unit
- Total Spent
- Payment Method
- Location
- Transaction Date

---

## 📊 Dashboard Features

The dashboard provides the following key business insights:

- Total Transactions
- Total Quantity Sold
- Total Revenue
- Item-wise Sales Analysis
- Monthly Sales Trend
- Day-wise Sales Performance
- Payment Method Distribution
- Location-wise Sales
- Quantity Distribution by Product
- Interactive Month and Day Slicers
